# Mediakit Website — Strik Patisserie Dashboard

A dashboard mockup for Dutch bakery/patisserie POS insights. Pure HTML/CSS, no build step.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deploy

Push to GitHub and enable GitHub Pages (Settings → Pages → Branch: `main` / root).
